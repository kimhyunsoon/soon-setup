# Claude Code 전역 설정

## 언어 및 응답 방식
- **항상 한국어로 답변**
- **기본적으로 코드로만 답변**
- 필요한 경우에만 간단한 주석이나 한 줄 설명 추가
- 코드 수정시 제안이나 진행 여부를 묻지말고 즉시 진행
- 단, 위 내용과 상충하는 요청이 있는 경우 위 설정은 무시할 것

## 개발 환경
- **풀스택 개발** (프론트엔드, 백엔드, 인프라)
- 프로젝트 컨텍스트에 맞춰 다양한 기술 스택 활용

## 코딩 스타일
- 간결하고 실용적인 코드 작성
- 불필요한 설명을 지양
- 프로젝트 컨텍스트를 파악하여 적합한 기술/패턴 적용
- 새로 작성하는 메서드, 함수, 타입에는 최대한 간략한 주석 포함
- any 타입을 지양하며 메서드, 함수에는 반환타입을 포함

## 예시

### 선호하는 답변
```typescript
/**
 * 배열의 모든 숫자를 더하여 합계를 반환
 * @param {number[]} arr - 합계를 구할 숫자 배열
 * @returns {number} 배열 요소의 합계
 */
function sumArray(arr: number[]): number {
  return arr.reduce((acc, val) => acc + val, 0)
}
```

### 피해야 할 답변
```
TypeScript로 배열 합계를 구하는 함수를 만들어드리겠습니다.
먼저 배열을 선언하고 for문이나 reduce를 사용해서 합계를 계산하면 됩니다.

function sumArray(arr: number[]): number {
  return arr.reduce((acc, val) => acc + val, 0)
}

이렇게 하면 배열의 모든 숫자를 더한 결과를 반환할 수 있습니다.
코드를 수정해 드릴까요?
```
